### Запуск проекта

`npm run dev` - запуск приложения в режиме разработки.

`npm run build` - сборка приложения в папку `dist`.

`npm run start` - сборка приложения и запуск (localhost:3000).

`npm run lint` - запуск eslint и stylelint.
